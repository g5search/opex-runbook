# Installation

  1. Clone the repo
    git clone https://github.com/g5search/opex-runbook.git
  2. Install NPM packages
    npm install
  3. Start Runbook Locally
    npm run docs:dev
  4. Access App for Local Development and/or Testing
    http://localhost:8080/opex-runbook/
